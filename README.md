# sysadmin-owntools
Herramientas creadas con python y bash para automatizacion, auditoria y mantenimiento de servidores Linux y Windows.

##
Herramienta 01: Auditor de Servidores Script en Python que analiza el uso de disco y detecta usuarios conectados en servidores Linux (RHEL/AlmaLinux). Genera logs automáticos para cumplimiento de auditoría.
